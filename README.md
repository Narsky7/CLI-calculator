# Simple CLI calculator

## Requirements

- C compilator

## How to run

Simply use script compile_script.sh

```
make

./bin/clicalc
```

## Usage
Right now you can use operations and sings as:
- +,-,*,/
- (,)
- sqrt(),ln(),sin()

```
$> 2 + 2 * 2
6
$> (2 + 3) * 5
25
$> sqrt(16) + 1
5
$> 10 / 4
2.5
$> exit
```
